# 下一步任务清单

更新时间：2026-02-16

## P0（优先）

1. 标签提取质量优化（规则 + 模型融合）
- 目标：降低误打标，提升命中率。
- 范围：
- 增加规则黑名单（噪声词、站点词、编码词）
- 增加“最小标签长度 / 最小词频 / 标题位置权重”
- 验收：
- 随机抽样 200 个视频，人工复核准确率提升到可接受区间（由你定义阈值）。

2. 候选标签评审效率优化
- 目标：高频候选可更快处理。
- 范围：
- 增加排序（按命中次数、时间）
- 增加“只看高频（>=N）”快速过滤
- 验收：
- 评审窗口内 500+ 候选项仍可流畅筛选与批量操作。

3. 主界面直接入口
- 目标：不进“标签提取窗口”也能直接进入候选评审。
- 范围：
- 在主窗口新增“候选评审”按钮
- 验收：
- 一键打开候选评审页面，评审后主界面可刷新状态文本。

## P1（可并行）

4. 标签提取结果可解释性
- 目标：知道每个标签从哪来（规则命中/模型命中/人工）。
- 范围：
- UI 增加来源标识
- 日志增加更清晰的事件字段
- 验收：
- 视频详情页可区分标签来源。

5. 检索能力增强
- 目标：通过标签、时长、日期快速定位视频。
- 范围：
- 增加组合筛选 UI（AND/OR）
- 验收：
- 常见场景（按标签 + 时长范围）2 步内可完成筛选。

## 暂缓（后续再做）

6. 在线大模型 API 辅助标签提取
- 先保持本地离线优先，后续再加可选 API 模式。

7. GPU 路径与模型切换策略自动化
- 在当前 CPU 可运行基础上，后续再补自动探测与切换。
